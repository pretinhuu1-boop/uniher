Banco de Dados
Cria uma cópia do banco de dados em data/backups/ com a data atual. Não altera nada no banco ativo. Executa PRAGMA integrity_check e foreign_key_check no SQLite. Verifica se os dados estão íntegros e se não há violações de chaves.
Sistema (Master)
Apaga o conteúdo do arquivo errors.log. Os erros futuros serão registrados normalmente. Apaga errors.log e server.log. O servidor continua funcionando e novos logs serão criados. Remove a pasta .next (cache do Next.js/Turbopack). O servidor vai recompilar tudo no próximo acesso — pode levar alguns segundos.
